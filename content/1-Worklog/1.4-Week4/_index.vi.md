### Kết quả đạt được tuần 4:

* Thành thạo quản lý AWS Resources bằng Tagging:
  * Tạo và quản lý tags cho resources
  * Lọc resources theo tags
  * Sử dụng CLI để quản lý tags
  * Tạo Resource Groups

* Hiểu sâu sắc về IAM Policy và Roles:
  * Tạo IAM Policies với hạn chế (Restriction Policies)
  * Tạo Limited Users với quyền cụ thể
  * Quản lý IAM Users, Groups, Roles
  * Cấu hình role switching với IP/Time constraints

* Nắm vững Audit và Logging Services:
  * CloudTrail - lỏn ghi tất cả API calls
  * Amazon Athena - truy vấn logs bằng SQL
  * Tạo và tới ưu hóa CloudTrail logs

* Quản lý khóa và mã hóa:
  * Dịch vụ quản lý khóa AWS (KMS)
  * Dữ liệu được mã hóa trên S3
  * Khóa truy cập được quản lý

* Thực hành thành công các lab:
  * Lab 27: Resource tagging và quản lý
  * Lab 30: IAM policy và hạn chế truy cập
  * Lab 33: IAM, KMS, CloudTrail, Athena integration
  * Lab 44: IAM Group, Users, Role switching
  * Lab 48: EC2, S3, IAM user và role đầy đủ

* Có khả năng bảo mật và kiểm tra độc lập các tài nguyên AWS.
* Nắm vững các best practices về security trong IAM.
  * Xem dịch vụ EC2
  * Tạo và quản lý key pair
  * Kiểm tra thông tin dịch vụ đang chạy


* Có khả năng kết nối giữa giao diện web và CLI để quản lý tài nguyên AWS song song.
