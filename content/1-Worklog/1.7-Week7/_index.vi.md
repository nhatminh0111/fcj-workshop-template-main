### Kết quả đạt được tuần 7:

* Nắm vững EC2 Connect RDP Client:
  * Kết nối từ xa đến các EC2 instance
  * Sử dụng Fleet Manager để quản lý nhiều instance
  * Khắc phục sự cố kết nối

* Thành thạo AWS Database Migration Service (Lab 43):
  * Kết nối đến các cơ sở dữ liệu nguồn (source databases) như SQL Server, Oracle
  * Cấu hình database
  * Tạo migration project và endpoint
  * Xử lý chuyển đổi schema (schema conversion)
  * Khắc phục sự cố trong các kịch bản di chuyển (migration scenarios)
  * Kiểm tra dữ liệu trên S3
  * Tạo các giải pháp di chuyển serverless
  * Ghi log và giám sát quá trình di chuyển

* Thành thạo AWS QuickSight Dashboard (Lab 73):
  * Tạo và xây dựng dashboard
  * Cải thiện chức năng của dashboard
  * Tạo dashboard tương tác (interactive)
  * Các best practices về trực quan hóa dữ liệu (visualization)

* Nắm được các quy trình phân tích dữ liệu (data analytics workflows):
  * Nguyên tắc thiết kế dashboard
  * Các best practices về trực quan hóa dữ liệu
  * Các kịch bản phân tích thực tế

* Có kinh nghiệm trong nghiên cứu dự án:
  * Phân tích các best practices của AWS
  * Hiểu các use case thực tế
  * Khả năng đánh giá các dự án cloud

* Có khả năng xây dựng các dự án AWS độc lập:
  * Kết hợp các dịch vụ AWS
  * Triển khai các giải pháp di chuyển (migration solutions)
  * Xây dựng các giải pháp phân tích dữ liệu
