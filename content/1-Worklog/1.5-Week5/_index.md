### Week 5 Achievements:

* Deep understanding of Database concepts and architecture:
  * Relational and Non-relational databases
  * Key-value stores
  * In-memory caches

* Mastered AWS database services:
  * Amazon RDS (Relational Database Service)
  * Amazon Aurora - database compatible with MySQL and PostgreSQL
  * Amazon Redshift - data warehouse
  * Amazon Elasticache - in-memory caching

* Successfully practiced deploying database infrastructure:
  * Created VPC and security groups
  * Created RDS instance with backup & recovery
  * Application deployment with database

* Understanding of AWS Data Pipeline services:
  * AWS Glue - extract, transform, load (ETL)
  * Amazon Kinesis - real-time data streaming
  * Amazon Athena - query data with SQL
  * Amazon QuickSight - data visualization

* Mastered cost optimization and tagging:
  * Setup cost allocation tags
  * Tracked usage and analyzed costs
  * Database optimization strategies

* Practiced Lambda automation:
  * Created Lambda functions
  * Integrated with SNS/Slack webhooks
  * Automated resource management (start/stop instances)

* Successfully completed advanced labs:
  * Lab 5: Complete database deployment with VPC
  * Lab 35: Data pipeline with Glue, Athena, QuickSight
  * Lab 40: Database cost analysis and optimization
  * Lab 22: Lambda automation with Slack integration

* Capable of deploying and managing complete database infrastructure.

  * View EC2 service
