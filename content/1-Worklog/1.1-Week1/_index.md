### Week 1 Objectives:

* Connect and get acquainted with members of First Cloud Journey.
* Create and configure an AWS Free Tier account.
* Understand IAM (Identity and Access Management) and access control.
* Master fundamental knowledge about Amazon VPC (Virtual Private Cloud).
* Practice deploying network models and EC2 connections in AWS environment.

### Week 1 Achievements:

* Successfully created an AWS Free Tier Account.

* Understood internship unit rules, regulations, and got acquainted with First Cloud Journey members.

* Mastered IAM (Identity and Access Management) concepts:
  * Create and manage IAM Groups
  * Create IAM Users with different permissions
  * Create IAM Roles and switch between roles
  * Implement secure access control

* Understood fundamental knowledge of Amazon VPC:
  * VPC structure, Subnets (Public/Private)
  * Internet Gateway, Route Table
  * Security Group, VPC Flow Logs
  * Enable and monitor VPC Flow Logs

* Successfully practiced deploying AWS network models:
  * Create VPC and public/private subnets
  * Configure Internet Gateway and Route Table
  * Create EC2 instances within VPC
  * Connect EC2 instances (Public and Private)
  * Deploy NAT Gateway to allow Private EC2 to access Internet
  * Use Reachability Analyzer to verify network connectivity

* Mastered advanced SSH connection tools:
  * EC2 Instance Connect Endpoint
  * VPN (Site-to-Site VPN)
  * Configure and use VPN environment securely
