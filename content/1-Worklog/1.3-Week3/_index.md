### Week 3 Achievements:

* Mastered Infrastructure as Code (IaC) with CloudFormation:
  * Created and managed templates
  * Automated AWS infrastructure deployment
  * Version control for infrastructure

* Comprehensive understanding of Migration Services:
  * VMware migration tools
  * Server Migration Service (SMS)
  * Virtual machine migration from On-premises to AWS
  * EC2 instance import/export capabilities

* Strong proficiency in AWS storage services:
  * Amazon S3 (Simple Storage Service)
  * EFS (Elastic File System)
  * Storage Gateway
  * Snow Family
  * Static Website configuration, CORS, Access Control

* Deep understanding of AWS Security:
  * Shared Responsibility Model
  * IAM (Identity and Access Management)
  * Amazon Cognito
  * AWS Organizations
  * AWS Identity Center
  * KMS (Key Management Service)
  * AWS Security Hub

* Successfully completed advanced labs:
  * Lab 19: CloudFormation, VPC Peering, NACLs
  * Lab 14: On-premises to AWS migration
  * Lab 25: EFS multi-AZ file systems, performance optimization
  * Lab 18: Security Hub enablement and assessment

* Capable of securely migrating infrastructure from On-premises to AWS.
* Understanding of security and compliance best practices in AWS.
  * View EC2 service
  * Create and manage key pairs
  * Check information about running services

* Acquired the ability to connect between the web interface and CLI to manage AWS resources in parallel.
