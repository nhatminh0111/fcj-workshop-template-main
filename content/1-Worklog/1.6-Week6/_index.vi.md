### Kết quả đạt được tuần 6:

* Nắm vững AWS CLI và Cloud Development:
  * AWS CloudShell - terminal trên web
  * AWS Management Console tools
  * AWS SDK - độc lập phát triển

* Thành thạo Cloud9 IDE:
  * Tạo và quản lý Cloud9 environments
  * Phát triển và debug code trên cloud

* Nắm vững AWS data analytics pipeline đầy đủ:
  * AWS Glue - ETL với interactive sessions và GUI
  * AWS Glue DataBrew - data preparation
  * AWS EMR - big data processing
  * Amazon Athena - SQL queries
  * Amazon Kinesis - real-time analytics
  * Amazon QuickSight - data visualization
  * AWS Lambda - serverless data serving
  * Amazon Redshift - data warehouse

* Hiểu sâu sắc về Advanced IAM:
  * Role-based access control across regions
  * Multi-region IAM policies
  * Tag-based access control
  * Policy analysis và validation

* Thành thạo Amazon DynamoDB:
  * DynamoDB basics và console operations
  * Backup và restore strategies
  * LADV - Advanced Design Patterns
  * LMR - Global Serverless Applications
  * LEDA - Event-Driven Architecture

* Thực hành thành công các lab phức hợp:
  * Lab 60: AWS CLI tools và management
  * Lab 70: Complete data analytics pipeline
  * Lab 28: Advanced IAM multi-region setup
  * Lab 72: End-to-end data processing
  * Lab 39: DynamoDB advanced patterns và serverless

* Có khả năng triển khai các big data solutions trên AWS.
* Khả năng xây dựng serverless applications với DynamoDB.
  * Tạo và quản lý key pair
  * Kiểm tra thông tin dịch vụ đang chạy

* Có khả năng kết nối giữa giao diện web và CLI để quản lý tài nguyên AWS song song.
