### Kết quả đạt được tuần 2:

* Hiểu rõ về hạ tầng toàn cầu của AWS:
  * AWS Regions và Availability Zones
  * Lợi ích của AWS Global Infrastructure
  * Cách chọn region phù hợp

* Nắm vững AWS Well Architected Framework:
  * Operational Excellence
  * Security
  * Reliability
  * Performance Efficiency
  * Cost Optimization

* Hiểu biết toàn diện về Amazon EC2:
  * Instance types và cách chọn phù hợp
  * AMI (Amazon Machine Image)
  * Storage options (EBS, Instance Store)
  * User data và Metadata
  * EC2 Auto Scaling
  * EFS, FSx, Lightsail, MGN

* Nắm được quản lý ngân sách và tối ưu hóa chi phí:
  * Tạo Cost Budget để theo dõi chi tiêu
  * Tạo Usage Budget để kiểm soát sử dụng
  * Hiểu về Reserved Instances (RI) Budget
  * Tìm hiểu về Savings Plans Budget
  * Cấu hình Virtual MFA Device

* Thực hành triển khai các dự án phức tạp:
  * Lab 13: Triển khai hạ tầng, backup và recovery
  * Lab 24: Sử dụng S3 và Storage Gateway
  * Lab 57: Cấu hình S3 Static Website, CloudFront, Bucket Versioning, Replication
  * Lab 10: Cấu hình CloudFormation, Security Group, Route 53 Hybrid DNS

* Có khả năng triển khai và quản lý hạ tầng AWS quy mô lớn.
* Hiểu rõ về AWS Global Infrastructure và cách tối ưu hóa chi phí trong môi trường production.
